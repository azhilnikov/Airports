This is an universal iOS application that downloads and displays information about airports around the world.
The application is written in Swift 3.0.1, built and tested in Xcode 8.2.1.
Gloss framework (version 1.2) is used to parse JSON objects and map them into model files.
Cocoapods was used to add Gloss library to the project.
Open Airports.xcworkspace file in Xcode and build the application.
